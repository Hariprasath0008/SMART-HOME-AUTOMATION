# CAPSTONE-PROJECT
This section contains the visual design and planning components of the Smart Home IoT Project.
All diagrams included here are meant to illustrate the overall system structure, workflow, and technical logic behind the implementation.


System Architecture	Shows how sensors, microcontrollers (Arduino/ESP8266), and the IoT platform communicate.
Use Case Diagram	Describes user interactions such as turning appliances ON/OFF or viewing sensor data.
Class Diagram	Represents logical relationships between different program classes/modules.
Circuit Diagram	Displays component connections — sensors, relays, power supply, and microcontroller setup.
